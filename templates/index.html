Welcome,

I created this website as my final project for CS50x 2020.

It's main purpose is to act as a "To-Do List" with a few extra functionalities.

You can use it to track your tasks and mark them as being complete or not.

You can use it to keep track of the events you register.

You can use it to keep track of warranties for things you buy.(I've always had trouble with this in real life)

At the bottom of the "Today" section there is a form where you can register the total money you spent/earned at the end of the day, there is also a little slider where you can rate your day from 0 to 100.

This is my first time building a website and a working backend server, I didn't really know what I was doing when I started but with the help of this course and a bunch of googling I managed to make a working version of what I set out to do initially.

I plan on using this little program for myself and I plan to keep adding features and fixing bugs in the future; everyone is welcome to use it too.

Thanks for checking it out!

Caution:

Please keep in mind that this is a beginner project and, while I tried to use some encryption protocols on the password, please try not to use any of the real passwords you use on other sites as I cannot guarantee their security.
